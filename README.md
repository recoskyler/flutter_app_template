# flutter_app_template Warehouse App

## Requirements

- Flutter 3.7.7
- Dart 2.19.4
- Android Virtual Device/Physical Android Device

## Development

1. Clone the project:

    ```bash
    $ git clone https://github.com/ardobirk/flutter_app_template.git
    ```

2. Generate the translations:

    ```bash
    $ flutter gen-l10n
    ```

3. Run the app:

    ```bash
    $ flutter run
    ```
